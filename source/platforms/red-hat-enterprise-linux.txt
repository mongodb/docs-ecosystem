Identity Management
-------------------

The Identity Management (IdM) solution in Red Hat Enterprise Linux
integrates Kerberos authentication, directory services, certificate
management, DNS, and NTP into a single service. The following
tutorials describe how to set up and configure MongoDB enterprise to
take advantage of these features when managing MongoDB deployments.
