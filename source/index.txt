=============================
MongoDB Drivers and Ecosystem
=============================

The MongoDB Ecosystem contains documentation for the drivers,
frameworks, tools, and platform services that work with MongoDB. For
the main MongoDB documentation, see the :manual:`MongoDB Manual </>`.

MongoDB ODM (Object-Document-Mapper)
------------------------------------

Mongoid is the officially supported ODM (Object-Document-Mapper)
framework for MongoDB in Ruby. For documentation, see `Mongoid
Documentation <https://docs.mongodb.com/mongoid/current/>`_.
